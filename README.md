# Hello-Spring
스프링 입문 - 코드로 배우는 스프링 부트, 웹 MVC, DB 접근 기술

### Spring

#### 웹 서버 실행
```shell
$ ./gradlew build
```

```shell
$ cd build/libs
```

```shell
$ java -jar hello-spring-0.0.1-SNAPSHOT.jar 
```
